# **08 — Memory System**

## **1. Summary Types**

Allowed values:

* `"persona"`
* `"goals"`
* `"content_strategy"`
* `"past_wins"`
* `"other"`

---

## **2. When to Update Memory**

* Immediately after onboarding
* After relevant user messages
* At periodic chat intervals

---

## **3. Memory in AI Calls**

Memory objects must be included in the **FACTS** block of every LLM request.
